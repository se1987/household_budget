## 概要
一人用の家計簿アプリです。
初学者が、フルスタックアプリ開発の学習のために作成しています。

## 使用技術
- フロントエンド：Next.js,Typscript,TailwindCSS
- バックエンド：Node.js(Express),prisma
- データベース：Postgres SQL

## 実装している機能
- 入出金の一覧表示
- 詳細画面の確認
- 収支入力
- 更新・削除
- 月間サマリー表示
